# 锤普降油记 | Bonk the Don: Oil Drop

Hammer Trump to make oil low.

A small satirical browser game: bonk the cartoon target, push the oil chart down, and enjoy the arcade chaos.

## Play Locally

Open `index.html` directly in a browser.

## Deploy

This repo includes a GitHub Pages workflow. Push to `main`, then GitHub Actions publishes the static files.

## Files

- `index.html` - page structure
- `style.css` - cartoon arcade visuals
- `game.js` - game loop, scoring, audio, and oil chart logic
